# 📊 FB_MJD Sales Analysis Dashboard

## 🧾 Project Overview  
The **FB_MJD Sales Analysis Dashboard** is an interactive Excel-based business intelligence solution designed to deliver actionable insights into sales performance, customer behavior, and service distribution.

This dashboard empowers stakeholders to monitor key performance indicators (KPIs), identify trends, and support data-driven decision-making in a clear and efficient manner.

---

## 🎯 Objectives  
- Analyze overall sales performance across multiple years  
- Track profitability and margin trends  
- Understand customer segmentation and behavior  
- Identify top-performing services and customers  
- Provide a centralized, user-friendly reporting tool
